
# [2025, Day 4](https://adventofcode.com/2025/day/4)

## Input:
A grid of line-seperated characters `.` or `@`, EX:

```
..@..@@
@.@.@@.
@@@@...
@....@@
```

## Part 1:
Determine which `@` have fewer than 4 other `@`s in the 8 adjacent spaces to them.

## Part 2:
Remove each `@` that have fewer than 4 other `@`s in the 8 adjacent spaces to them. Replace them with a `.`. Repeat this until no more `@` can be removed.
