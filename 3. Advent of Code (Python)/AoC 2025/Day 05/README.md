
# [2025, Day 5](https://adventofcode.com/2025/day/5)

## Input:
Two sets of data: first, a line-seperated set of ID ranges. Then, a blank line, followed by a line-seperated set of ID numbers. EX:
```
4-7
10-20
15-30

4
6
8
20
40
```

## Part 1:
Determine which ID numbers in the second set of data are within the given ID ranges in the first set.


## Part 2:
Determine the entire set of ID numbers contained within the given set of ID ranges.
