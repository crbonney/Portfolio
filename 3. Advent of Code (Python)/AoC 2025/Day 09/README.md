
# [2025, Day 9](https://adventofcode.com/2025/day/9)


## Input:
A line-seperated set of 2D coordinates that form the vertices of a concave polygon, The last point loops back and joins to the first. EX:
```
8,2
10,2
10,8
6,8
6,4
8,4
```

## Part 1:
Determine the largest possible rectangle that contains two vertices in the dataset.


## Part 2:
Determine the largest possible rectangle that is *completely contained within* the polygon described in the dataset.
