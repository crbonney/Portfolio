
# [2025, Day 1](https://adventofcode.com/2025/day/1)

## Input:
A set of strings describing the turning of a 100-digit dial that starts at 50. Input is given as a list of directions for rotation direction (L,R) and an integer for rotation quantity, ex:
```
L3
R30
R33
L423
```

## Part 1:
Calculate the number of times the dial finishes a movement pointing at 0.

## Part 2:
Calculate the number of times the dial touches 0 (including mid rotation).
