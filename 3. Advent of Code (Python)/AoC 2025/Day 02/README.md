
# [2025, Day 2](https://adventofcode.com/2025/day/2)

## Input:
A comma-seperated list of integer ranges of ID numbers, ex:
```
10-23,300-432,999-1000
```

## Part 1:
Determine which ID numbers in *all* of the given ranges are in the form of a sequence of digits repeated exactly twice (ex: 11, 1212, 987987)

## Part 2:
Determine which ID numbers in *all* of the given ranges are in the form of a sequence of digits repeated any number of times (ex: 111, 131313, 456456456456)
